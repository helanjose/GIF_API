# GIF_API
#Work flow

 * Theme of this small application is animals.
 * There is an array of animals in the application
 * For each animal in the array there is a button in the UI
 * Whenever a button is clicked 10 gif vedeos of that particular      animal is displayed on the screen with it's ratng
 * Each time when a new button is clicked instead of overwriting      the previous gifs ,new gifs will be prepended so that when we      scroll we can see all gifs 
 * There is an option for adding new animal name which will appear     as button 
 * When newly created buttons click it will also show the             corresponding animal gif
